La URI (Identificador de recursos uniforme)

La URI es una cadena de caracteres para Identificar un recurso en una red.

  • Puede ser una URL, URN o ambos
  • Definida en RFC 3986
  • Usada para hiperenlaces en la WWW
  • Sistema para nombrar y localizar recursos en la Web

Clases de URI

  • URL: Localiza el recurso.
  • URN: Nombra el recurso.

Descripción de las partes

Parte Descripción
Esquema Protocolo de acceso al recurso
Parte Jerárquica Ruta al recurso (autoridad y path)
Cadena de consulta Parámetros de búsqueda (query)
Fragmento Sección específica del documento

Ejemplos de URIs

  • ftp://ftp.is.co.za/rfc/rfc1808.txt
  • http://www.ietf.org/rfc/rfc2396.txt
  • mailto:John.Doe@example.com
  • https://fis.epn.edu.ec/index.php?option=com_content&view=article&id=137&Itemid=294
